# EFF Analysis: State-Mandated 3D Printer Censorship Will Fail but Harm Users

**TL;DR:** EFF explains why proposed state laws requiring "print blockers" on 3D printers won't prevent firearm manufacturing but will create censorship infrastructure that harms legitimate users and manufacturers.

## Key Points

- Multiple US states proposing laws to mandate "print blocking" technology on 3D printers and CNC machines
- Laws would require printers to scan files against government-maintained database of prohibited designs
- EFF argues technical implementation is fundamentally flawed - printers use G-Code instructions, not 3D models
- Proposed restrictions would lock users into vendor software and create ongoing compliance costs
- Many printer computers lack computational power to render G-Code back into 3D models for comparison
- Laws would criminalize circumventing scanning or loading alternative software on owned devices
- Existing noncompliant printers could not be legally resold in regulated states
- Technology creates foundation for broader 3D-print censorship platform in other jurisdictions
