# EFF: States Push DRM-Style 'Print Blocking' Requirements for 3D Printers

**TL;DR:** Three U.S. states propose mandating 3D printer manufacturers install algorithmic censorship to block 'gun-like' shapes, with criminal penalties for bypassing the restrictions.

## Key Points

- Three U.S. states proposed laws requiring 3D printer manufacturers to install 'print blocking' algorithms
- Proposed systems would scan prints for 'forbidden shapes' deemed too gun-like by vendors
- Criminal penalties proposed for bypassing restrictions or reselling unrestricted printers
- EFF argues this creates DRM-style vendor lock-in similar to other anti-competitive practices
- Laws would restrict printers to manufacturer-only software, blocking open source alternatives
- Bambu Labs recently forced restrictive firmware updates limiting OrcaSlicer compatibility
- Targeted 'ghost gun' production already illegal in proposing states
- Technical implementation faces cat-and-mouse detection problems covered in EFF's part 2
