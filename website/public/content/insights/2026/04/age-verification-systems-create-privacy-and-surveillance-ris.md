# Age Verification Systems Create Privacy and Surveillance Risks as Regulations Spread

**TL;DR:** New age verification laws worldwide are forcing platforms to implement identity-linked systems that threaten anonymous internet access and create surveillance infrastructure.

## Key Points

- Age verification requirements spreading globally beyond traditional "checkbox honor system"
- Two main approaches: age estimation (biometric inference) vs age verification (ID documents)
- Age estimation uses facial/voice analysis but suffers from accuracy issues and demographic bias
- Age verification requires government IDs, creating identity-linked records even when documents deleted
- Australia's age verification trial showed higher error rates for darker skin tones and Indigenous groups
- Adult content sites like Pornhub blocking entire states rather than implementing ID requirements
- Most systems create persistent records linking verified users to specific content access
- Both approaches undermine pseudonymous internet use and create surveillance infrastructure
