# Russians Navigate State Censorship vs Security Trade-offs During Internet Restrictions

**TL;DR:** Citizen Lab researcher warns Russians are accepting state-controlled apps over secure alternatives during internet shutdowns, prioritizing immediate access over privacy when seeking critical information.

## Key Points

- Senior Citizen Lab researcher Ksenia Ermoshina analyzed Russian censorship evasion patterns
- Russian authorities actively blocking apps, websites, and mobile internet access
- Some Russians successfully circumventing censorship while others settle for state-controlled alternatives
- Citizens prioritize immediate information access over security during critical moments
- Parents especially willing to trade privacy for ability to check on children's safety
- Pattern suggests state censorship strategy may be succeeding in shifting user behavior
- Highlights tension between operational security and urgent communication needs
