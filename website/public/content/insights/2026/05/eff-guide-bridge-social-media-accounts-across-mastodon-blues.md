# EFF Guide: Bridge Social Media Accounts Across Mastodon, Bluesky, and Threads

**TL;DR:** EFF published a comprehensive guide on using bridging tools like Bridgy Fed to connect accounts across federated social platforms, enabling cross-platform posting and reducing platform lock-in.

## Key Points

- EFF released tutorial on bridging Mastodon, Bluesky, and Threads accounts for cross-platform posting
- Focuses on Bridgy Fed as primary bridging service, with alternatives like Fedisky and RSS Parrot mentioned
- Implements POSSE philosophy (Post Own Site Syndicate Elsewhere) to reduce platform dependency
- Mastodon users can bridge to Bluesky by following @bsky.brid.gy@bsky.brid.gy
- Threads requires enabling Fediverse sharing in settings, with one-week delay for Bluesky bridging
- Bluesky users bridge to Fediverse by following @ap.brid.gy account
- Some instances may block bridging tools or Threads accounts entirely
- Enables users to reach audiences across platforms without maintaining multiple accounts
