# Apple & Google Deploy E2E Encrypted Cross-Platform Messaging via RCS

**TL;DR:** Apple's iOS 26.5 enables end-to-end encryption for RCS messages between iPhone and Android, making encrypted cross-platform messaging the default for billions of users.

## Key Points

- Apple released iOS 26.5 with end-to-end encrypted RCS messaging support
- Cross-platform messages between iPhone Messages app and Google Messages now encrypted by default
- Feature requires carrier support for both RCS and encrypted messaging
- Implementation uses GSMA RCS Universal Profile 3.0 with Messaging Layer Security protocol
- Neither Google, Apple, nor carriers can access message contents with this update
- Metadata collection likely continues, making Signal still preferable for high-risk conversations
- Cloud backups may remain unencrypted unless Advanced Data Protection enabled
- Feature currently in beta on Apple devices due to carrier rollout dependencies
