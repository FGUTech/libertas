# Tails 7.7.3 Emergency Release Fixes Critical Kernel Vulnerability

**TL;DR:** Tails released emergency update fixing 'Dirty Frag' kernel vulnerability that could allow privilege escalation and user deanonymization, plus Tor Browser and client security patches.

## Key Points

- Emergency Tails 7.7.3 release addresses critical security vulnerabilities
- Linux kernel updated to 6.12.86 to fix 'Dirty Frag' vulnerability
- Dirty Frag could allow applications to gain admin privileges in Tails
- If exploited with other vulnerabilities, could lead to full system compromise and deanonymization
- Tor Browser updated to version 15.0.12 with security fixes
- Tor client updated to 0.4.9.8 addressing additional vulnerabilities
- Tails team reports no known exploitation of vulnerability in practice
- Automatic upgrades available from Tails 7.0 or later
