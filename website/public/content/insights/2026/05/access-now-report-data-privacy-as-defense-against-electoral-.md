# Access Now Report: Data Privacy as Defense Against Electoral Disinformation in Latin America

**TL;DR:** Access Now analysis shows personal data collection fuels political microtargeting and disinformation, recommending strict data protection enforcement during elections across Latin America.

## Key Points

- Access Now published analysis linking data collection to disinformation campaigns targeting Latin American elections
- Report identifies microtargeting as key threat: using personal data to deliver manipulative political content
- Generative AI has reduced costs and technical barriers for creating deepfakes and bot networks
- Social media algorithms use personal data profiles to enable political manipulation and voter suppression
- Recommends data protection authorities collaborate with electoral authorities to combat coordinated disinformation
- Proposes disabling microtargeting tools for political content during election periods
- Calls for algorithmic impact laboratories to monitor platform compliance with recommendations
