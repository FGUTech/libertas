# Tails 7.8 Fixes Critical Security Vulnerabilities That Could Enable Deanonymization

**TL;DR:** Tails 7.8 patches critical Linux kernel vulnerabilities that could allow attackers to gain admin privileges and potentially deanonymize users, while removing pre-installed Thunderbird.

## Key Points

- Tails 7.8 fixes multiple security vulnerabilities in Linux kernel and haveged that could allow privilege escalation
- Vulnerabilities could enable attackers to gain full control of Tails system and deanonymize users
- Updates Tor Browser to version 15.0.14
- Removes pre-installed Thunderbird email client to address security update lag issues
- Thunderbird can still be installed as additional software for automatic updates
- Users with Persistent Storage automatically get Thunderbird added to additional software list
- Automatic upgrades available from Tails 7.0 or later versions
