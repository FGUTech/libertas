# Access Now: Privacy Laws Key to Fighting Disinformation in Latin America Elections

**TL;DR:** Access Now argues that strict data protection laws are essential to combat personalized disinformation campaigns targeting voters across upcoming Latin American elections.

## Key Points

- Access Now published analysis linking data collection practices to disinformation effectiveness
- Microtargeting uses personal data to deliver personalized political propaganda and false information
- AI-generated content (deepfakes) and bots amplify targeted disinformation campaigns
- Social media platforms' business models enable data harvesting for political manipulation
- Report focuses on upcoming elections in Argentina, Mexico, Brazil, Colombia, and Peru
- Recommends banning sensitive data processing that reveals political preferences
- Calls for cooperation between data protection and electoral authorities
- Proposes disabling microtargeting tools during electoral periods
