# Tails 7.7.2 Emergency Release Fixes Critical Privilege Escalation Vulnerability

**TL;DR:** Tails released emergency update to patch Copy Fail vulnerability in Linux kernel that could allow attackers to gain admin privileges and potentially deanonymize users.

## Key Points

- Emergency Tails 7.7.2 release fixes critical Linux kernel vulnerability
- Copy Fail (CVE in kernel 6.12.85) enables privilege escalation attacks
- Vulnerability could allow applications to gain administration privileges
- Potential attack chain: exploit app vulnerability + Copy Fail = full system control
- Successful exploitation could lead to user deanonymization
- Tails team reports no known real-world exploitation yet
- Automatic upgrades available from Tails 7.0+, manual upgrade option provided
