# Citizen Lab Exposes How Ad-Tech Data Enables Government Location Surveillance

**TL;DR:** New Citizen Lab investigation reveals how Webloc surveillance system leverages ad-tech data to enable government tracking, with ICE using it to circumvent legal protections.

## Key Points

- Citizen Lab investigated Webloc, a geolocation surveillance system using ad-based data
- System enables governments to access location data collected by advertising ecosystem
- ICE reportedly uses this technology to circumvent legal protections for citizens
- Ad-tech infrastructure creates vast surveillance opportunities for state actors
- Current internet ecosystem designed around personal data collection for advertisers
- Legal protections being bypassed through commercial data purchases
- Disabling location sharing provides only partial protection from tracking
