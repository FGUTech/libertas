# Tails 7.4.1 Emergency Release Fixes Critical OpenSSL Vulnerabilities in Tor

**TL;DR:** Tails released emergency update fixing critical OpenSSL vulnerabilities that could allow malicious Tor relays to deanonymize users.

## Key Points

- Emergency Tails 7.4.1 release addresses critical OpenSSL library vulnerabilities (DSA 6113-1)
- Vulnerabilities could potentially allow malicious Tor relays to deanonymize Tails users
- OpenSSL updated to version 3.5.4, Tor client updated to 0.4.8.22
- Tails team reports no known exploitation of these vulnerabilities in practice
- Additional fixes include Gmail authentication in Thunderbird and Wi-Fi settings UI improvements
- Automatic upgrades available from Tails 7.0 or later versions
- Known issue: Tor Browser homepage incorrectly displays version 7.4 after upgrade
