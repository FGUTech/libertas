# Tails 7.4 Adds Persistent Language Settings, Drops BitTorrent Support

**TL;DR:** Tails 7.4 introduces persistent language/keyboard settings and removes BitTorrent downloads due to security concerns with v1 to v2 transition.

## Key Points

- New feature allows saving language and keyboard layout settings to USB stick for automatic application on restart
- Language/keyboard data stored unencrypted on USB to enable easier Persistent Storage passphrase entry
- Updated Tor Browser to 15.0.4 and Thunderbird to 140.6.0 for latest security patches
- Linux kernel updated to 6.12.63 for improved hardware support and security
- BitTorrent download support removed due to security concerns with v1 format during transition to v2
- Fixed critical bugs including VeraCrypt desktop crashes and Kleopatra GPG file handling
- Automatic upgrades available from Tails 7.0+ with manual upgrade fallback option
