# New York Proposes Mandatory 3D Printer Surveillance for Firearm Detection

**TL;DR:** New York's budget bill would require all 3D printers sold in the state to include 'blocking technology' that scans files for potential firearm blueprints and refuses to print flagged items.

## Key Points

- NY budget bill S.9005/A.10005 mandates 'blocking technology' in all 3D printers sold in state
- Required software/firmware must scan print files with 'firearms blueprint detection algorithm'
- Printers would refuse to print anything flagged as potential firearm or component
- Bruce Schneier warns this creates same problems as DRM - controlling general-purpose computers
- Measure targets maker community, educators, and small manufacturers statewide
- Technical approach fundamentally flawed - algorithms can be bypassed or circumvented
- Creates precedent for government-mandated surveillance in manufacturing tools
- Threatens user sovereignty over personally-owned devices and equipment
