# Signal Rolls Out Secure Backups Feature After Years of User Requests

**TL;DR:** Signal launches opt-in secure backups in Android beta, addressing the top user request to preserve encrypted message history when phones are lost or damaged.

## Key Points

- Signal begins rolling out secure backups feature after extensive development
- Feature is opt-in and currently available in Android beta version only
- Addresses most common user request for message history preservation
- Enables recovery of messages, media, and documents if phone is lost/damaged
- iOS and Desktop versions planned for near future release
- Represents significant UX improvement for encrypted communications adoption
- Maintains Signal's security model while solving usability gap
