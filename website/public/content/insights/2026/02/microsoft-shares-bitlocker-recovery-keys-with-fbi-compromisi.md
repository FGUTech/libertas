# Microsoft Shares BitLocker Recovery Keys with FBI, Compromising Disk Encryption

**TL;DR:** Microsoft routinely provides BitLocker backup keys to law enforcement like the FBI, undermining the privacy protection of Windows' built-in disk encryption for users storing keys in Microsoft accounts.

## Key Points

- Microsoft routinely hands over BitLocker recovery keys to law enforcement agencies including FBI [Forbes]
- BitLocker is Windows' built-in full-disk encryption, with Device Encryption variant on Windows Home
- Recovery keys automatically backed up to Microsoft accounts on Windows Home devices
- Windows Pro/Enterprise users can choose where to store recovery keys (USB, file, print, Active Directory)
- TPM 2.0 chip protects encryption keys on modern systems, but recovery keys remain accessible to Microsoft
- Device seizure while powered off provides strong protection unless recovery key is compromised
- Users relying on Microsoft account backup effectively give government access to encrypted data
