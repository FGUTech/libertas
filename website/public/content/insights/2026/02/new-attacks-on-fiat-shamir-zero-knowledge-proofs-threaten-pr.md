# New Attacks on Fiat-Shamir Zero-Knowledge Proofs Threaten Privacy Systems

**TL;DR:** Researchers found practical attacks against Fiat-Shamir-based zero-knowledge proof systems, potentially compromising privacy protocols that rely on these cryptographic foundations.

## Key Points

- Khovratovich, Rothblum and Soukhanov published 'How to Prove False Statements' showing attacks on Fiat-Shamir
- Targets GKR15 proof system and similar interactive protocols converted to non-interactive using Fiat-Shamir heuristic
- Attack exploits 'weak challenges' - specific challenge values that allow proving false statements
- Affects proof systems used in privacy-preserving applications and blockchain protocols
- Interactive versions remain secure, but non-interactive Fiat-Shamir conversions vulnerable
- Hash functions like SHA-3 replacing random oracles create exploitable gaps in security
- Threatens zero-knowledge systems used for private transactions and anonymous credentials
