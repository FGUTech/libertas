# AI Tools Create New Censorship & Information Access Risks

**TL;DR:** AccessNow analysis shows LLMs pose availability risks through data deletion, service disruptions, and content refusals. DeepSeek censors Tiananmen Square topics, while open-source models enable bias removal.

## Key Points

- AccessNow examines how AI/LLMs impact information availability and system access
- German professor lost two years of data when ChatGPT deleted all chats after settings change
- AI agents can disrupt availability by wiping data or making unauthorized system changes
- Chinese DeepSeek R1 refuses to answer questions about Tiananmen Square and sensitive topics
- Perplexity AI successfully retrained open-source DeepSeek model to remove censorship
- Google Bard allegedly refused all questions containing Gaza/IDF keywords
- Closed-source models from Google/OpenAI lack transparency in their refusal mechanisms
- Open-source AI models enable customization and reduce dependency on corporate gatekeepers
