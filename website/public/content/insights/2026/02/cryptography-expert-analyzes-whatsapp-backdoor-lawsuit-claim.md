# Cryptography Expert Analyzes WhatsApp Backdoor Lawsuit Claims

**TL;DR:** Class action lawsuit alleges WhatsApp can read all encrypted messages despite E2E claims. Cryptography expert Matthew Green examines technical feasibility and evidence quality of backdoor allegations.

## Key Points

- Quinn Emanuel law firm filed class action alleging WhatsApp lacks real end-to-end encryption despite public claims
- Lawsuit claims 'whistleblowers' revealed Meta can read all WhatsApp messages except for some celebrities since April 2016
- Bloomberg reports US authorities investigating Meta based on these same allegations
- Elon Musk and Pavel Durov (who operate competing messaging apps) amplified the claims online
- WhatsApp serves ~3 billion users with Signal protocol-based E2E encryption enabled by default since 2014
- Cryptography expert Matthew Green notes complaint lacks solid evidence for backdoor claims
- Government pressure against WhatsApp encryption has been intense from US, UK, Australia, India, and EU
- If allegations proved true, would represent major corporate coverup affecting half the planet's communications
