# Tails 7.0 RC2 Released with Debian 13, Updated Privacy Tools

**TL;DR:** Tails privacy OS releases second release candidate based on Debian 13, updating core freedom tech tools like Tor, Electrum Bitcoin wallet, and OnionShare file sharing.

## Key Points

- Tails 7.0~rc2 is second release candidate, official release planned September 18 or October 16
- First Tails version based on Debian 13 (Trixie) and GNOME 48
- Increases minimum RAM requirements from 2GB to 3GB (affects <2% of users)
- Updates Tor client to 0.4.8.17 and Linux kernel to 6.12.41 for better hardware support
- Updates Electrum Bitcoin wallet from 4.3.4 to 4.5.8
- Updates OnionShare anonymous file sharing from 2.6.2 to 2.6.3
- Updates KeePassXC password manager from 2.7.4 to 2.7.10
- Removes aircrack-ng wireless security tool and other legacy software
- Known issue: slower boot times to be fixed in final release
