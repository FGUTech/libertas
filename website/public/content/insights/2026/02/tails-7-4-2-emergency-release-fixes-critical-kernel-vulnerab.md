# Tails 7.4.2 Emergency Release Fixes Critical Kernel Vulnerabilities

**TL;DR:** Tails issued emergency update fixing Linux kernel vulnerabilities that could allow privilege escalation and potential deanonymization of users.

## Key Points

- Emergency release addresses DSA 6126-1, multiple critical Linux kernel security vulnerabilities
- Vulnerabilities could allow applications to gain admin privileges and potentially deanonymize users
- Linux kernel updated to version 6.12.69 to patch security flaws
- Attack vector requires chaining with other exploits, making it complex but possible for advanced adversaries
- Tails team notes governments or hacking firms could potentially exploit these vulnerabilities
- Additional fixes include Wi-Fi settings, Electrum wallet crashes, and language persistence issues
- Automatic upgrades available from Tails 7.0+, manual upgrade option provided for compatibility issues
