# Access Now Report: AI Tools Systematically Undermine Data Confidentiality and Privacy

**TL;DR:** Major AI platforms like ChatGPT and Meta AI are compromising user privacy through data breaches, lack of encryption, and forced integration that breaks E2EE promises.

## Key Points

- Access Now published comprehensive analysis of AI security vulnerabilities threatening confidentiality
- DeepSeek exposed publicly accessible database with chat histories and sensitive user data [Wiz Research]
- OpenAI suffered breach exposing names, emails, location data via third-party analytics provider
- Meta AI integration in WhatsApp breaks E2EE by default - summaries pass through Meta servers unencrypted
- Urban Cyber Security VPN extensions promising 'AI protection' actually harvested 8 million users' LLM data
- New York Times lawsuit seeks access to 20+ million private ChatGPT conversations for copyright case
- Most AI chatbots lack end-to-end encryption as standard feature, leaving chat histories exposed
- Emerging 'agentic AI' with system-level access creates new attack vectors via prompt injection
- Report frames issues using CIA triad (Confidentiality-Integrity-Availability) security model
