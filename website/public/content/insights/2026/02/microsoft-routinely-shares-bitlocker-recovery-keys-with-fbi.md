# Microsoft Routinely Shares BitLocker Recovery Keys with FBI

**TL;DR:** Proton explains BitLocker encryption following revelations that Microsoft regularly provides BitLocker recovery keys to law enforcement agencies like the FBI.

## Key Points

- Microsoft routinely hands over BitLocker backup keys to law enforcement agencies including FBI [Forbes]
- BitLocker is Windows' built-in full-disk encryption, with Device Encryption as simplified version on Windows Home
- Device Encryption automatically backs up recovery keys to Microsoft accounts by default
- Windows Pro/Enterprise BitLocker allows users to choose where recovery keys are stored (USB, file, printed copy)
- TPM 2.0 chip provides hardware-based protection but keys still recoverable via Microsoft accounts
- Protection strong against theft/loss when powered off, but vulnerable to legal process targeting Microsoft
- Demonstrates risks of relying on proprietary encryption with centralized key backup systems
