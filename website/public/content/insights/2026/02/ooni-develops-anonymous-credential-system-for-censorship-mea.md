# OONI Develops Anonymous Credential System for Censorship Measurement Privacy

**TL;DR:** OONI is integrating anonymous credentials into OONI Probe to increase trust in censorship measurements while preserving user privacy and anonymity.

## Key Points

- OONI developing anonymous credential system for OONI Probe censorship measurement tool
- Goal is to enrich measurement data with probe properties while preserving user privacy
- System aims to increase trust in OONI's global censorship measurement dataset
- Part of broader effort to enhance security without requiring user identification
- OONI published comprehensive academic literature review of anonymous credential systems
- Critical for protecting activists and researchers measuring internet censorship
- Could set precedent for privacy-preserving verification in freedom tech tools
