# Silent.Link: Bitcoin-Native eSIM Service Bypasses SIM-Swap Attacks and State Censorship

**TL;DR:** Silent.Link offers privacy-focused eSIM data/SMS services paid with Bitcoin, eliminating SIM-swap risks while bypassing state firewalls in China and Dubai's VoIP restrictions.

## Key Points

- Silent.Link provides global eSIM-only mobile data and SMS services paid with Bitcoin
- No personal information collected - users can purchase without even providing email
- Authentication via backup codes (similar to Bitcoin seed phrases) eliminates SIM-swap attack vectors
- Reportedly bypasses Chinese firewall and Dubai VoIP restrictions
- Founded by Bitcoin community members inspired by Matt Odell, profitable and self-funded
- Automated roaming across countries without carrier negotiations or overcharges
- No outgoing SMS or voice calls - focuses on data and incoming authentication messages
- Built on BTCPay Server stack with Lightning support for Bitcoin-native payments
