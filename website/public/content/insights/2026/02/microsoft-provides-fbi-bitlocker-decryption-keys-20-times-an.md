# Microsoft Provides FBI BitLocker Decryption Keys ~20 Times Annually

**TL;DR:** Microsoft gives FBI access to BitLocker encryption keys about 20 times per year via court orders when users store keys on Microsoft's servers for convenience.

## Key Points

- Microsoft provides FBI BitLocker decryption capability ~20 times annually in response to court orders
- Users can store encryption keys locally but Microsoft recommends cloud storage for convenience
- Cloud-stored keys enable account recovery but expose users to law enforcement subpoenas
- Demonstrates how centralized key storage creates surveillance backdoors in mainstream encryption
- BitLocker appears secure but has architectural surveillance vulnerability when using default settings
- Shows importance of local key storage for actual encryption security
- Highlights gap between perceived and actual privacy protection in enterprise encryption tools
