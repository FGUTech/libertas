# India Civil Society Updates OONI Website Censorship Test Lists

**TL;DR:** The Bachchao Project and OONI collaborated to update the website list used for monitoring internet censorship in India, strengthening civil society's ability to track government blocking.

## Key Points

- The Bachchao Project partnered with OONI to review India's website test list in June 2025
- Updated list used by OONI Probe to monitor website censorship across India
- Workshop involved experts updating specific categories of tested websites
- Part of broader effort to strengthen civil society engagement in internet freedom monitoring
- Test list maintained by Citizen Lab contains URLs regularly tested for blocking
- Enhanced monitoring capabilities provide better visibility into government censorship patterns
- Represents organized approach to documenting digital rights violations in India
