# Cryptographers Find Practical Attacks on Fiat-Shamir Zero-Knowledge Proofs

**TL;DR:** New research reveals vulnerabilities in Fiat-Shamir heuristic used in ZK proof systems, showing how attackers can potentially prove false statements in widely-used cryptographic protocols.

## Key Points

- Paper by Khovratovich, Rothblum and Soukhanov demonstrates practical attacks on Fiat-Shamir heuristic
- Fiat-Shamir transforms interactive ZK proofs into non-interactive ones using hash functions
- Vulnerability allows proving false statements in systems that appear secure
- Attack exploits gap between theoretical security models and real-world implementations
- Affects verifiable computation systems commonly used in blockchain and privacy applications
- Research highlights ongoing tension between practical crypto deployment and theoretical guarantees
- Issue stems from replacing random oracles with actual hash functions in practice
