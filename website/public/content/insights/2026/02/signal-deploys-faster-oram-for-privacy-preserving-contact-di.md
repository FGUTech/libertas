# Signal Deploys Faster ORAM for Privacy-Preserving Contact Discovery

**TL;DR:** Signal improved their contact discovery system using advanced ORAM techniques and enclaves, laying groundwork for usernames and enhanced phone number privacy controls.

## Key Points

- Signal enhanced Contact Discovery Service Infrastructure (CDSI) with new ORAM implementation
- System lets users find Signal contacts without revealing social graphs to Signal servers
- Uses secure enclaves and oblivious memory access patterns to protect privacy
- New implementation is faster and more efficient than previous version
- Technical foundation enables upcoming username feature and phone number privacy controls
- Represents advancement in privacy-preserving identity infrastructure
- Demonstrates scalable approach to private contact discovery for secure messaging
