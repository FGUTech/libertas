# OONI Publishes Complete Security Audit Report for Censorship Measurement Tools

**TL;DR:** OONI publicly released a comprehensive security audit report of its internet censorship measurement platform, conducted by Radically Open Security with OTF support.

## Key Points

- OONI published full penetration test report conducted by Radically Open Security
- Security audit ran from August 5 to October 18, 2024 with Open Technology Fund support
- Crystal box penetration test evaluated core components of OONI's censorship measurement software
- Complete transparency approach - full detailed report made publicly available
- Audit focused on ensuring high security standards for tools used by activists and researchers
- Part of OONI's broader commitment to secure, high-quality free software development
- Critical for user safety given OONI's use in repressive environments worldwide
