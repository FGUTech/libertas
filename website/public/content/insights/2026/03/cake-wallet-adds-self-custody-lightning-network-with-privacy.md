# Cake Wallet Adds Self-Custody Lightning Network With Privacy Defaults

**TL;DR:** Cake Wallet integrated Bitcoin Lightning Network with self-custody and privacy-first design using Breez SDK and custom Spark implementation, avoiding common Lightning privacy leaks.

## Key Points

- Cake Wallet launched Lightning Network support with full self-custody via Breez SDK and Spark
- Custom Spark implementation prevents address embedding in Lightning invoices
- Transaction data not published to public explorers by default
- Wallet already supports advanced Bitcoin privacy features like Silent Payments and Payjoin
- No Lightning node management required for users while maintaining self-custody
- Added social payments feature 'Birdpay' for sending crypto to X.com usernames
- Positions Cake among few wallets offering Lightning with strong privacy defaults
