# EFF Fights Legal Challenge That Could Make Website Embedding Illegal

**TL;DR:** EFF filed amicus brief defending 'server test' that protects website embedding from copyright liability. Publisher wants to make link embedding directly liable for infringement.

## Key Points

- Emmerich Newspapers challenges 20-year 'server test' precedent in Fifth Circuit appeal
- Server test holds that whoever controls hosting server bears primary copyright liability, not those who embed links
- Publisher argues embedding constitutes 'displaying' content, making embedders directly liable for infringement
- EFF warns this would make common internet activities legally risky and destabilize web architecture
- Millions of websites embed external content for fonts, streaming, customer support, compliance tools
- Additional claim: URL shorteners could violate DMCA by altering copyright management information
- District court rejected publisher's theories; case now on appeal
- Outcome could fundamentally change how information flows online
