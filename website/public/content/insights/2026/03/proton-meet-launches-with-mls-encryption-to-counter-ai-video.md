# Proton Meet Launches with MLS Encryption to Counter AI Video Surveillance

**TL;DR:** Proton launched Meet, an end-to-end encrypted video conferencing service using MLS protocol, addressing AI-powered surveillance risks in mainstream platforms like Zoom and Google Meet.

## Key Points

- Proton Meet launched as encrypted alternative to Zoom, Google Meet, and Microsoft Teams
- Uses Messaging Layer Security (MLS) protocol for end-to-end encryption at enterprise scale
- Built on WebRTC with Selective Forwarding Units (SFU) for scalability without P2P IP exposure
- Addresses AI surveillance threat from Big Tech companies recording/analyzing meeting data
- Features zero-knowledge meeting authentication via SRP - Proton cannot access meetings
- Offers web browser access without account required, plus dedicated mobile/desktop apps
- Free tier available with enterprise-grade features and admin controls
- Distributed global server network for low-latency, high-reliability communications
