# Tails 7.6 Adds Automatic Tor Bridge Discovery for Censorship Circumvention

**TL;DR:** Tails 7.6 introduces automatic Tor bridge configuration based on user region, making censorship circumvention easier for activists and journalists in restrictive networks.

## Key Points

- Tails 7.6 adds automatic Tor bridge discovery through 'Connect to Tor automatically' feature
- New 'Ask for a Tor bridge based on your region' option helps users in censorship-heavy areas
- Uses same technology as Tor Browser 11.5's connection assistant from July 2022
- Bridge discovery uses Tor Project's Moat API with domain fronting to disguise connections
- Replaces KeePassXC password manager with GNOME Secrets for better desktop integration
- Updates include Electrum 4.7.0, Tor Browser 15.0.8, and Thunderbird 140.8.0
- Improved firmware packages enhance support for newer hardware including Wi-Fi and graphics
- Automatic upgrades available from Tails 7.0+ with manual upgrade fallback option
