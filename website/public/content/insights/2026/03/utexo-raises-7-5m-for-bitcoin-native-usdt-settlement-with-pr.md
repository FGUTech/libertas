# Utexo Raises $7.5M for Bitcoin-Native USDT Settlement with Privacy Features

**TL;DR:** Utexo secured $7.5M led by Tether to build Bitcoin-native USDT settlement infrastructure with encrypted transactions and sub-second settlement, abstracting Lightning Network complexity behind a single API.

## Key Points

- Utexo raised $7.5M seed round co-led by Tether, Big Brain Holdings, and Portal Ventures
- Building Bitcoin-native USDT settlement infrastructure with privacy-preserving features
- Encrypts all on-chain transactions to hide counterparties and wallet addresses
- Abstracts Lightning Network and RGB protocol complexity behind single API layer
- Enables sub-second USDT settlement with predictable fees independent of network congestion
- Targets payment processors, exchanges, and wallets handling large USDT volumes
- Tether CEO Paolo Ardoino emphasizes Bitcoin's role as 'open and resilient settlement infrastructure'
- Infrastructure allows wallets to offer free USDT transactions while maintaining custody workflows
